# Week 03

This week covers API development, frontend integration, and working with LLMs (Large Language Models).

## Subfolders
- `api_project/`: FastAPI backend with Gemini LLM integration.
- `api_frontend/`: Next.js frontend for interacting with the backend.
- `llm_api/`: Jupyter notebooks and experiments with LLM APIs.
- `slides/`: Lecture slides and resources.

See each subfolder for more details.
